                            //check if string is palindrome
                            function checkForPalindrome(userString) {
                            
                                //taco cat
                                userString = userString.toLowerCase();
                            
                                //remove spaces and special characters
                                let regex = /[^a-z0-9]/gi;
                                userString = userString.replace(regex,"");
                            
                                //reverse string
                            
                                let revString = [];
                                let returnObj = {};
                            
                                for (let index = userString.length - 1; index >= 0; index--) {
                                    revString += userString[index];
                            
                                    if(revString == userString) {
                                        returnObj.msg = "Nice! You entered a palindrome!"
                                    }
                                    else {
                                        returnObj.msg = "Sorry! You did not enter a palindrome."
                                    }
                                }
                                returnObj.reversed = revString;
                            
                                return returnObj;
                            }
                            
                            //display a message to the string
                            function dislpayMessage(returnObj) {
                                document.getElementById('alertHeader').innerHTML = returnObj.msg;
                                document.getElementById('msg').innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                                document.getElementById('alert').classList.remove('invisible');
                            }
The code is structured in three functions
function getValue()

is the controller function where the program begins by getting the user's input. First, the program makes sure that the invisible class on the alert-ID is added so that the user doesn't see the alert message for checking if their string is a palindrome before the app starts. Next, the program gets the userString from the page via getElementById. Next, it checks the palindrome by calling the following logic function for checkForPalindrome. It also calls the displayMessage function so that the message ultimately is displayed for the user.

function checkForPalindrome

is the logic functino where the program checks if the the string is a palindrome. First, it calls for the userString to be taken in as all lower case letters so that there is no confusion for proper nouns or capital letters. Next, the program employs regex so that only letters and integers are "read" by the program, special characters like "!" or "?" are not counted in the string for the palindrome checker. Then, the program begins the logic to reverse the string. Reversing a string begins with declaring revString as an empty array and returnObj as an empty object. Next, it uses a for-loop to loop through the string backwards. An if/else statement is used to determine if the reversed string is the same as the user's input. If it isn't, a display message says "Sorry! You did not enter a palindrome." If the string IS reversed and it's the same as the user input, it will display "Nice! You entered a palindrome" The logic function concludes by declaring returnObj that's reversed is equal to the reversed string and then returning the returnObj.

function displayMessage>

is the view function. This function sends the user a "view" to see. They will see an alert header with the return object message by removing the invisible class on the alert-ID. The code uses a template literal and embedded expression to display the standard message with the addition of the user's string that's reversed as well as the "Nice!" or "Sorry!" for whether it's a palindrome or not.
